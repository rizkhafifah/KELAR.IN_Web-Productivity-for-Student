## Deskripsi Karya
Nama "Kelar.in" diambil dari peran aplikasi sebagai asisten/enabler, alat yang membantu untuk "Ngelarin" segala urusan akademik. 

Kelar.in adalah solusi teknologi pendidikan (EdTech) yang menangani masalah yang sering dialami mahasiswa yaitu Burnout dan Prokrastinasi. Dengan menggabungkan Behavioral Psychology (Gamifikasi) dan Artificial Intelligence (Gemini AI), kami menciptakan ekosistem yang mengubah manajemen tugas yang membosankan menjadi interaktif dan menyenangkan.

## The Problem (The "Why")
Berdasarkan analisis pain points mahasiswa saat ini:

1. Aplikasi produktivitas yang kaku malah meningkatkan tingkat stres (Kortisol).

2. Seringnya melakukan Prokrastinasi/penundaan dalam mengerjakan tugas.

3. Banyak mahasiswa menghabiskan waktu untuk membaca ulang, bukan menguji pemahaman.

4. Produktivitas seringkali mengorbankan kesehatan fisik dan mental.

## Key Innovation (The "How")
1. AI Study Lab (Active Recall Generator)

Fitur ini menggunakan Multimodal AI untuk memproses PDF, gambar catatan tangan, atau teks mentah menjadi kuis interaktif secara instan. Ini memaksa otak melakukan Active Recall, yang mana merupakan metode belajar paling efektif menurut sains.

2. Gamified Reward System

Setiap tugas yang diselesaikan menghasilkan Candy Points dan Daily Streaks. Sistem leveling ini dirancang untuk memicu pelepasan Dopamin positif, mengubah persepsi "beban tugas" menjadi "misi pencapaian" yang seruu.

3. Health Integration

Kelar.in adalah planner yang mengintegrasikan Water Intake dan Mood Tracking dalam satu dasbor. Kami percaya mahasiswa yang sehat adalah mahasiswa yang produktif.

## Tech Stack
Framework: React 19.

Style: Tailwind CSS 4 

AI Engine: Google Gemini API Integration untuk analisis konten dan chatbot mentor.

Data Handling: PDF.js & Mammoth.js untuk ekstraksi data dokumen.

## Roadmap/Rencana kedepannya
1. Integrasi Kalender (Google/Outlook) secara dua arah.

2. Fitur "Study Room" Multiplayer untuk kolaborasi antar mahasiswa.

3. Analisis prediktif berbasis AI untuk mendeteksi tanda-tanda Burnout lebih awal.

## Installation 
1. Clone & Install:

git clone <repo-url> && npm install

Buat file .env dan masukkan GEMINI_API_KEY dari Google AI Studio.

2. Deployment:
